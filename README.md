# cohort_analytics_ecommers
- Основная цель — измерить удержание клиентов (Retention Rate), рассчитать пожизненную ценность (LTV) по когортам и дать практические рекомендации маркетинговому отделу для повышения эффективности бюджета.
- Источник данных: Online Retail II (UCI) — публичный датасет на Kaggle.
- Главный инсайт: Основной доход клиенты приносят в первый месяц. Дальнейшие покупки незначительно увеличивают.
- Рекомендации: расширить ассортимент  магазина и предлагать товары не только к новогодним/рождественским праздникам, а и в межсезон, сосредоточить бюджет на удержании в первый месяц (welcome-кампании, промокод на второй заказ) и отдельно работать над монетизацией лояльной аудитории (программа лояльности, персонализация).
